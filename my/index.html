  • 많은 div 태그 사용으로 인해 가독성이 떨어짐
  • style을 적용해야 하기 때문에 class 이름이 난잡함

시맨틱 태그 장단점

장점

  1. 접근성 향상: 스크린 리더와 같은 보조 기술을 사용하는 사람들은 시맨틱 태그를 통해 페이지의 구조와 콘텐츠를 이해하는 데 도움을 받을 수 있습니다.
  2. SEO 향상: 시맨틱 태그는 검색 엔진이 웹 페이지의 콘텐츠를 이해하는 데 도움을 줍니다. 이는 검색 결과 순위를 높이는 데 기여할 수 있습니다.
  3. 유지 관리 용이: 시맨틱 태그는 코드의 가독성을 높여 웹 페이지의 유지보수를 용이하게 합니다.

단점

  1. 접근성 저하: 시맨틱 태그를 사용하지 않으면 보조 기술을 사용하는 사람들이 페이지의 콘텐츠를 제대로 이해하지 못할 수 있습니다.
  2. SEO 저하: 검색 엔진은 페이지의 콘텐츠를 제대로 이해하지 못할 수 있어, 검색 결과 순위에 영향을 미칠 수 있습니다.
  3. 유지 관리 어려움:유지 관리 어려움: 시맨틱 태그를 사용하지 않으면 웹 페이지의 코드 가독성이 떨어지며, 이는 유지보수의 어려움으로 이어질 수 있습니다.
